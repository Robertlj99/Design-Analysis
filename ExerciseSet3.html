Since f(n) = O(g(n)) there exists integers c and n0 such that f(n) ≤ c * g(n), when n0 ≤ n 

If 2f(n) = O(2g(n)) then the following cases would be true:

case 1: f(n) = g(n) = n

-this adheres to the rule f(n) = O(g(n))

-this gives us the formula 2f(n) = O(2g(n)), which is clearly true

case 2: f(n) = 10n, g(n) = n

-this adheres to the rule f(n) = O(g(n))

-this gives us the formula 210n = O(2n), which can be proven false using the following contradiction

---if 210n = O(2n) there exists integers c and n0 such that 210n ≤ c * 2n == 29n ≤ c which is clearly not true

Here we see that f(n) = O(g(n)) is not always true or false

Knowing that it is sometimes true we can examine the inequality for 2f(n) = O(2g(n))

-- 2f(n) ≤ c * 2g(n) == 2f(n) - g(n) ≤ c

Intuitively this will hold true when f(n) - g(n) ≤ 0 == f(n) ≤ g(n)
